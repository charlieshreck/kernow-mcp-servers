# Knowledge MCP

Consolidated MCP for semantic search, graph queries, wiki, and task management.

## Components

- **Qdrant**: Semantic search (runbooks, docs, entities, decisions, events)
- **Neo4j**: Graph queries (dependencies, impact analysis)
- **Vikunja**: Task/kanban management
- **Outline**: Wiki/documentation
